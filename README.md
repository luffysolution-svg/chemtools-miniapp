# 材料化学科研工具箱

一个专业的化学计算微信小程序，提供单位转换、XRD分析、电化学计算、溶度积计算等多种科研工具。

**当前版本**: v3.4.0  
**最后更新**: 2025-10-11

---

## ✨ 特色功能

### 🧮 基础计算工具
- **单位转换**: 长度、质量、体积、能量、浓度等多种单位转换
- **pH计算**: 酸碱溶液pH值计算
- **溶液配制**: 浓度转换、稀释计算
- **摩尔计算**: 摩尔质量、物质的量计算

### 🔬 高级计算工具
- **XRD分析**: 布拉格方程、d值计算、支持多种X射线源
- **电化学计算**: 参比电极转换、能斯特方程、电势计算
- **溶度积计算**: 50+种难溶盐数据、活度系数修正
- **络合计算**: 配合物稳定常数、条件稳定常数

### 📊 材料数据库
- **元素周期表**: 118种元素完整数据
- **半导体材料**: 52种材料，包含带隙、晶体结构等
- **光谱数据**: XPS、Raman、UV-Vis数据

### 🎨 用户体验
- **搜索历史**: 自动保存搜索记录，快速重复搜索
- **计算历史**: 完整的历史记录，支持筛选和导出
- **数据导出**: 支持TXT、Markdown、CSV、JSON格式
- **使用帮助**: 完整的帮助系统，包含数据来源和FAQ
- **骨架屏**: 优化加载体验
- **手势交互**: 下滑关闭模态框
